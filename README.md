# FROSTLINE

Landing page moderna para una marca de ropa y equipamiento de nieve, enfocada en diseño limpio, experiencia visual y responsive design.

## Tecnologías

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Características

* Diseño responsive
* Showcase de productos
* Sección multimedia con video
* Reseñas de usuarios
* Formulario de contacto
* Interfaz moderna y minimalista
